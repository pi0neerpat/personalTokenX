> A Superfluid-based social token experiment

## Prerequisites

- node >=12

- yarn >=1.15

## Install

```sh
yarn install
```

## Build the contracts

```bash
cd contracts
yarn build
```

## Run the app

```bash
# in root
yarn rw dev
```
